# RoutinCare FE

Frontend project cho RoutinCare, su dung React + Vite.

## Run project

Yeu cau:
- Node.js 18+
- npm 9+

Lenh chay:

```bash
npm install
npm run dev
```

Build production:

```bash
npm run build
npm run preview
```

Lint:

```bash
npm run lint
```
